# linux shell命令的学习和使用

## shell的基本格式

*#!/bin/bash*  那种shell脚本解释器。

## linux 中的shell解释器有哪些

* 在linux中有Bourne shell(/usr/bin/sh或者/bin/sh)

* Bourne Again Shell (/bin/bash)

* C shell (/usr/bin/csh)

* ...

shell开头都要加上解释器的类型

## shell脚本的基本格式

```
#!/bin/bash
echo "Hello World !"
```
由类型解释器语法与脚本语法两部分组成
